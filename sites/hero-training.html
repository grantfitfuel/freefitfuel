Hero Training

A 12-month, progressive path for strong, durable, enduring humans. Clear phases, smart overload, and habits that fit real life — for any age.

Strength (progressive overload) Endurance (Z2 → Tempo → Threshold) Mobility (hips/shoulders/spine) Recovery (sleep, deloads, nutrition)

How it works

  • Phases (I–IV) — 12 weeks each. Basics → Build → Peak → Sustain. Marathon as the year-end stamina goal (walk/run ramps included).
  • Overload — add a little load, a rep, or a set when form is clean. Deload every 4th week.
  • Form first — tap any workout to open a YouTube form search in a new tab.
  • Track locally — log kg + reps per set; data is stored in IndexedDB (survives normal cookie/history clears).

Tip: snap a private progress photo each month for yourself. Consistency + evidence = motivation.

Strength overload — the simple rule

Add load or 1–2 reps per set only when the last rep is clean: neutral spine, full range, no jerk. If the last rep gets messy, keep the load and build control first. Every 4th week: reduce volume by ~30% (deload) to bank adaptation.
