Phase 2 — Progression (Weeks 9–16)

We keep technique crisp and apply planned overload. Conditioning adds the first threshold pieces while the long run grows.

Overview Weekly Structure Strength Progression Conditioning Progression Mobility Checkpoint

Overview

  • 3 strength days (upper/lower split or full-body), 2–3 conditioning (Z2 + 1 threshold), mobility daily.
  • Overload: add load or total work on 1–2 lifts per week; rotate which lift you push.
  • Every 4th week: deload ~20–30% volume or intensity (feel fresh by week’s end).

Weekly Structure (example)

  • Mon — Strength Upper + mobility
  • Tue — Z2 40–60 min
  • Wed — Strength Lower + mobility
  • Thu — Threshold: e.g., 3 × 8 min “comfortably hard” (90–95% of 10k effort) with easy jog recoveries
  • Fri — Mobility + optional accessory strength (lighter)
  • Sat — Long Z2 (build 10–20% over Phase 1’s longest
  • Sun — Rest + mobility

Strength Progression (Weeks 9–16)

Upper Focus

  • 1-Arm DB Press — 4×6–8 /side (slower down, strong lockout)
  • 1-Arm DB Row — 4×8–10 /side
  • Concentration Curl — 3×8–12 /side
  • DB Overhead Triceps — 3×10–12

Lower Focus

  • FFESS / Split Squat — 4×6–8 /side
  • DB RDL — 4×6–8
  • DB Hip Thrust — 3×8–10
  • Calf Raise — 3×10–15

Overload plan: prioritise adding load on the first big lift of each day; on secondary lifts add a rep or a set. If reps stall, reduce tempo (3–1–1) for a week to spur adaptation.

Conditioning Progression (Weeks 9–16)

  • Z2 volume: lift to ~180–240 min/week by week 16 if recovery is good.
  • Threshold day (1×/week): e.g., 3 × 8 min @ comfortably hard with 3–4 min easy between. Build to 4 × 8 or 3 × 10 as tolerated.
  • Long Z2: add 10% most weeks; every 3–4th week, reduce volume ~20% to freshen.

If running hurts, substitute cycling or brisk incline walking. Keep the *effort* matched (not the speed).

Mobility (daily 8–15 min)

  • 90/90 Hips • 3–5 min
  • Lat Openers (banded) • 2–3 min /side
  • Ankle Dorsiflexion Drills • 2–3 min

Checkpoint & Next

  • Strength: main lifts up 5–10% from Phase 1 end? Good. If not, deload a week, then climb again.
  • Stamina: long Z2 extended without niggles? If yes, you’re ready for Phase 3 build.
  • Recovery: sleep steady, appetite normal, mood stable — green lights to progress.

When ready, continue to Phase 3 (Performance) or hold this phase longer if life is busy — sustainability first.

Important Note

This is general guidance. If you have conditions or injuries, speak to a professional. Stop any session that causes sharp pain, dizziness, chest pain, or unusual shortness of breath.
