Phase III — Build (Weeks 17–28)

Progressive overload on primary lifts (lower reps, heavier), introduce **threshold** engine work, keep daily mobility. Click any exercise for a YouTube form search. Log sets, reps, and kg; use the rest timers.

Weekly Structure

  • 3× Strength: Day A • Day B • Day C (lower rep ranges)
  • 2–3× Engine: one long Z2, one threshold (tempo/intervals), optional easy spin/walk
  • Daily Mobility: 8–12 minutes (hips + T-spine)
  • Long run builds toward ~16–18 km by the end of the phase

Day A — Squat + Push

  • Back Squat — 5×3–5 (heavy, leave 1–2 reps in reserve)
  • DB Bench Press — 4×6–8
  • Bulgarian Split Squat — 3×8–10/leg
  • Side Plank — 3×40–60 s/side

Day B — Hinge + Pull

  • Romanian Deadlift — 5×5
  • Barbell Row — 4×6–8
  • Half-Kneeling DB Press — 3×8–12/side
  • Farmer Carry — 3×40–60 m

Day C — Power + Accessories

  • KB Swing — 6×15–20
  • Incline DB Press — 3×8–12
  • Lat Pulldown / Assisted Pull-up — 3×8–12
  • Copenhagen Plank — 2–3×20–30 s/side

Calisthenics Skill (sprinkle 2×/wk)

  • Pull-up Progression — 4–6 sets of 2–5 strict
  • Wall Handstand Practice — 6–10×20–30 s
  • Pike Push-ups — 3–4×6–10

Threshold Session (1×/wk)

“Comfortably hard”; breathe through nose if possible, sentences broken.

  • Tempo Run — 10 min easy → 2–3×8 min tempo (3 min easy) → 10 min easy
  • Intervals — 6–8×2 min strong (2 min easy)

Easy Engine + Long Run

  • Z2 Easy — 30–60 min
  • Long run progresses by ~10% toward 16–18 km by week 28

Mobility — Hips

  • 90/90 Hip Flows — 2–3 min
  • Couch Stretch — 2×60–90 s/side
  • Adductor Rockbacks — 2×10–12

Mobility — T-Spine

  • Open Book — 10/side
  • Thread the Needle — 8/side
  • Prone Swimmers — 2×6–8
