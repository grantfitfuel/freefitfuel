Workouts

Simple weekly split with global difficulty, clear sets & reps, and a form library. Pick a level, then open each day.

Difficulty:
Push Day — Chest • Shoulders • Triceps (Beginner)
  • Push-up — 3×8–12 (incline if needed)
  • Dumbbell Floor Press — 3×8–12
  • Lateral Raise — 3×12–15
  • Band Triceps Press-down — 3×10–14
  • Optional: DB Thruster — 2×10 (light)
Push Day — Chest • Shoulders • Triceps (Intermediate)
  • DB Bench Press — 4×6–10
  • Arnold Press — 3×8–12
  • Cable/Band Fly — 3×12–15
  • Overhead Band Triceps — 3×10–14
  • DB Thruster — 3×8–10
Push Day — Chest • Shoulders • Triceps (Advanced)
  • DB Bench Press — 5×5–8 (slow eccentric)
  • DB Incline Press — 3×8–10
  • Lateral Raise mechanical drop set — 3 rounds
  • Close-grip Push-up — 3×AMRAP
  • DB Thruster — 4×6–8 heavy-controlled
Pull Day — Back • Rear Delts • Biceps (Beginner)
  • Doorway Row — 3×8–12
  • Band Lat Pulldown (door anchor) — 3×10–14
  • Chest-Supported DB Row — 3×8–12
  • Zottman Curl — 3×10–12
Pull Day — Back • Rear Delts • Biceps (Intermediate)
  • Band-Assisted Chin-ups — 4×6–10
  • Chest-Supported DB Row — 4×8–10
  • Renegade Row — 3×8–10 / side
  • Zottman Curl — 3×10–12
Pull Day — Back • Rear Delts • Biceps (Advanced)
  • Chin-ups (weighted if strong) — 5×5–8
  • Chest-Supported DB Row — 4×6–8 heavy
  • Renegade Row — 4×10–12 total
  • DB Hammer→Zottman finisher — 2×12 each
Legs Day — Quads • Glutes • Hamstrings • Calves (Beginner)
  • Step-ups — 3×8–12 / leg (box height mid-shin)
  • DB Romanian Deadlift — 3×8–12
  • Bodyweight Squat — 3×12–15
  • Calf Raise — 3×12–20
  • Side Plank — 2×20–30s / side
Legs Day — Quads • Glutes • Hamstrings • Calves (Intermediate)
  • DB Back-foot Elevated Split Squat — 4×6–10
  • DB Romanian Deadlift — 4×8–10
  • Step-ups — 3×10–12 / leg (higher box)
  • Calf Raise — 4×12–20
  • Side Plank — 3×30–45s / side
Legs Day — Quads • Glutes • Hamstrings • Calves (Advanced)
  • DB Front Squat — 5×5–8
  • DB Romanian Deadlift — 4×6–8 heavy-controlled
  • Walking Lunges — 3×20 steps total
  • Standing Calf Raise — 4×15–25
  • Side Plank (loaded) — 3×45–60s / side
Full-Body Day Combination
  • Man Maker — 3×6–8
  • DB Clean & Press — 3×6–8
  • Wall Handstand Hold (or plank) — 3×20–40s
