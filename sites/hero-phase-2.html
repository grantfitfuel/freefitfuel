Phase II — Build

Weeks 9–16. We add volume and a little intensity: hills/tempo for stamina, progressive loads in strength, and sharper calisthenics skills. Still plenty of Z2 to keep the engine purring.

Run / Z2 + Tempo Strength Calisthenics Recovery

Run Engine — Z2 + Tempo / Hills

2–3 runs/week. Keep Z2 conversational; add one quality day.

Quality Day (choose one)

  • Tempo: 10 min Z2 warm-up → 2×8–12 min comfortably-hard (T pace) with 3–4 min easy between → 10 min easy
  • Hills: 10 min Z2 warm-up → 6–10× 45–60s uphill hard, easy walk down → 10 min easy

Effort guide: you can speak in short phrases, not full sentences on the work sections.

Long Z2 Run

Build 45–75 min, easy nose-breathing pace. Walk/jog fine.

If not running: brisk walk, ruck, cycling, or row Z2 same durations.

Strength — Progress the Load

Front-rack Reverse Lunge · DB RDL · 1-Arm Row · DB Push Press

Lower-Focus

  • Front-rack Reverse Lunge — 4×6–8/side
  • DB RDL — 4×6–8
  • Calf Raises — 3×12–15
  • Core: Bird Dog — 3×10/side

Upper-Focus

  • 1-Arm DB Row — 4×8–10/side
  • DB Push Press — 4×6–8
  • Incline Push-ups or Floor Press — 3×8–12
  • Core: Side Plank — 3×30–45s/side

Progression: aim to add 1 rep per set each week, then add load when you hit the top of the range.

Calisthenics — Level Up

Diamond Push-ups · Inverted/Under-table Rows · Arch hang → Pull-up
  • Push-ups: 5×sub-max (progress to diamond/elevated feet as strong)
  • Rows / Assisted Pull-ups: 5×6–10
  • Hollow / Arch rocks: 3×20–30s each

Recovery & Mobility

  • 10–15 min mobility most days (hips, calves, T-spine)
  • Breath: Box 4-4-4-4 or 4-7-8 before bed
  • Fuel: protein each meal, carbs around quality or long sessions, hydrate

Weekly Template (example)

  • Mon — Strength Lower + mobility
  • Tue — Z2 30–45 min
  • Wed — Strength Upper + calisthenics
  • Thu — Tempo or Hills
  • Fri — Easy Z2 20–30 min + mobility
  • Sat — Long Z2 45–75 min
  • Sun — Rest / Breathwork
