Phase I — Foundation (Weeks 1–4)

Goal this phase: learn the movements, build easy Zone 2 engine, and hit 10-minute daily mobility. Long run eases up to ~6–8k. Keep it fun and consistent.

Weekly rhythm: 3× strength days • 2–3× engine days • daily 10-min mobility. Check things off to rack XP (stored locally).

Day A — Lower + Push

  • Goblet Squat 4×6–10 — YouTube
  • DB Romanian Deadlift 4×8–12 — YouTube
  • DB Floor Press 4×8–12 — YouTube
  • Side Plank 3×30–45s/side — YouTube

Day B — Hinge + Pull

  • KB/DB Deadlift 5×5–8 — YouTube
  • One-Arm Row 4×8–12/side — YouTube
  • Half-Kneeling Press 3×8–12/side — YouTube
  • Carry (Front/Suitcase) 3×40–60m — YouTube

Progression: add a rep or a tiny bit of load weekly while keeping perfect form.

Easy Engine (Z2)

2–3 sessions / week • 20–45 mins easy pace. Nasal breathing, can chat in full sentences.

  • Run / brisk walk / bike / row — How to find Z2
  • End-of-phase long effort ≈ 6–8 km run equivalent

Tempo Taster

1 session / week • 20–30 mins incl. 2×5-min “comfortably hard”.

  • Warm-up 10 min easy → 2×5 min strong with 3 min easy recovery
  • Cool-down 5–10 min

Boss Fight Preview

Optional check: 5k run/row time or “Row Alternative”. Aim to finish feeling like you could do more.

Recovery Hygiene

  • Sleep: 7–9h target (consistent times)
  • Food: protein 1.6–2.2g/kg; carbs around engine days
  • Hydration: bodyweight(kg) × 30–35 ml baseline
  • Breathing: 5–10 min coherent (in 5 / out 5) — YouTube
