Phase 1 — Foundation (Weeks 1–8)

Movement quality, base strength, and the aerobic habit (Zone 2). We start small and build steadily.

Overview Weekly Structure Strength Plan Conditioning Plan Mobility How to Progress

Overview

  • 3 strength days (full-body) + 2–3 conditioning (Z2) + daily 6–12 min mobility.
  • Overload rule: add a little (load, reps or a set) only when all sets are clean with stable technique.
  • Older trainees: consider 48–72h between hard strength sessions, and keep long Z2 truly easy.

Weekly Structure (example)

  • Mon — Strength A + mobility
  • Tue — Z2 30–45 min + mobility
  • Wed — Strength B + mobility
  • Thu — Z2 30–45 min (walk/run/cycle/row) + mobility
  • Fri — Strength A (lighter or technique focus) + mobility
  • Sat — Optional Z2 30–60 min (easy) or rest + mobility
  • Sun — Rest + mobility

Strength Plan (Weeks 1–8)

Day A

  • Goblet Squat — 3×8–10
  • DB Floor Press — 3×8–10
  • DB Row — 3×10–12 /side
  • Split Squat — 2–3×8 /side
  • Dead Bug — 2–3×8 /side (slow)

Day B

  • DB RDL — 3×8–10
  • 1-Arm DB Press — 3×8–10 /side
  • Concentration Curl — 2–3×10–12 /side
  • DB Hip Thrust — 3×10–12
  • Side Plank — 2–3×20–40s /side

Progressive Overload: if you hit the top of the rep range across all sets with perfect form, next time increase the load ~2.5–5% (or add 1–2 reps if you can’t add load).

Conditioning Plan (Weeks 1–8)

  • Goal: 150–240 min/week of true Z2 by week 8, built gradually.
  • Start: 2–3 × 20–30 min (walk, run/walk, cycle, row) — conversation pace.
  • Build: add ~5–10 min total weekly; every 3rd or 4th week ease back ~20%.

Rowing alternative if no machine: Inverted/Table Rows won’t replace Z2 cardio, but can be used for pulling volume on strength days.

Mobility (daily 6–12 min)

  • World’s Greatest Stretch • 2–3 min
  • Couch Stretch • 2–3 min /side
  • T-spine openers • 2–3 min

How to Progress

  • Strength: when you complete all sets at the upper rep target with crisp form, add load next time (2.5–5%). If load options are coarse, alternate adding reps then load.
  • Conditioning: extend one Z2 session by 5–10 min. Keep it truly easy.
  • Recovery: if fatigued, hold loads steady a week and sleep more — consistency wins.

Important Note

General guidance only. If you have health conditions or injuries, consult a professional. Stop any session that causes sharp pain, dizziness, chest pain, or unusual shortness of breath.
