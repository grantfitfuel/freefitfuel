Engine — Z2 Streak (3–4×/week)

How

  • Start at 20–30 min conversational pace (you can nose-breathe, talk in sentences)
  • Add +5 min when it feels easy 2 sessions in a row
  • Optional: one short tempo taste (8–12 min “comfortably hard”) in week 3–4

How to run each test
  • Run — 1 mile: warm up 5–10 min easy. Start stopwatch, cover 1 mile at best sustainable pace, stop at finish. Record time.
  • DB Rows — 12 min AMRAP: choose a moderate DB. Set 12-min timer. Alternate: 10/10 rows per minute (or sets you can keep tidy). Count total reps. Strict form beats speed.
  • Table/Ring Rows — 12 min AMRAP: set a stable table/rings low. 8–12 clean reps per set; rest as needed. Count total reps.
